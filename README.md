# Exoplanet Habitability System (EHS)

## Overview

The **Exoplanet Habitability System (EHS)** is a data science and machine learning project that analyzes NASA's exoplanet dataset to identify planets with characteristics that may support habitability.

The project involves data collection, preprocessing, exploratory data analysis (EDA), feature engineering, and predictive modeling to evaluate the habitability potential of discovered exoplanets.

---

## Objectives

- Collect exoplanet data from NASA.
- Validate and clean the dataset.
- Analyze planetary and stellar characteristics.
- Engineer meaningful habitability features.
- Train machine learning models.
- Predict and rank potentially habitable exoplanets.

---

## Dataset

Source: NASA Exoplanet Archive

Current dataset contains approximately **6,300+ confirmed exoplanets** with planetary and stellar parameters.
